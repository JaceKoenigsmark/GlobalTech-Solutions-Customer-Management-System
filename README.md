# GlobalTech Solutions Customer Management System

## Overview

The GlobalTech Solutions Customer Management System is a Python-based business operations project that uses dictionaries, lists, and aggregation techniques to manage client accounts, service offerings, projects, budgets, and recommendations. The system simulates how a consulting or technology company could organize customer data and generate business insights. :contentReference[oaicite:0]{index=0}

This project demonstrates practical Python programming skills often used in CRM systems, service operations, and business analytics.

---

## Features

### Customer Management
- Store customer records using dictionaries
- Retrieve customer details by ID
- Update customer contact information
- Validate customer profiles

### Service Management
Tracks hourly rates for services such as:

- Web Development
- Data Analysis
- Cybersecurity
- Cloud Consulting
- Technical Support

### Project Management
- Assign projects to customers
- Store hours, budgets, and service type
- Track project status
- Summarize customer workloads

### Financial Analytics
- Calculate project costs
- Total budgets
- Average project values
- Highest and lowest budget projects

### Business Intelligence
- Service usage frequency
- Customer spending totals
- Active customer list
- Service tier classification
- Personalized service recommendations
